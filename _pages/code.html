The following links contain code to algorithms developed and used in our papers:

  • Convolutional neural networks for time series forecasting: The following link contains the code used in the paper "Conditional time series forecasting with convolutional neural networks". This code allows to train a convolutional neural network based on the WaveNet architecture for the forecasting of time series. It can be applied to e.g. financial time series, weather forecasting.

  • Training neural networks for solving PDEs: The following link contains the code used in the paper "Optimally weighted loss functions for solving PDEs with neural networks". This code allows to train a neural network using Tensorflow for learning the solution to a wide variety of PDEs.
